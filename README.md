# Analyze Subscription Data

Проект анализа пользовательских подписок: предобработка данных, очистка, работа с пропусками,
обработка дубликатов, построение словарей категорий и подготовка данных для аналитики.

## Состав проекта
- `analizdannyh.ipynb` — ноутбук с анализом данных
- `case_data.xlsx` — данные по платежам
- `doc_views.csv` — данные просмотров документов

## Навыки, использованные в проекте
- Python (pandas, numpy)
- Очистка данных
- Обработка пропусков
- Обработка дубликатов
- Работа с типами данных
- Построение связей между таблицами
- Создание словаря категорий

## Результат
Данные приведены к чистому, аналитическому виду и готовы для построения дашбордов/моделей.
